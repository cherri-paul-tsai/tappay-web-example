# TapPay Web Example

此專案是 TapPay SDK Get Prime 的範例
我們會介紹使用以下幾種 Get Prime 的方式

1. Direct Pay - iframe
2. Direct Pay - TapPay Fields
3. Apple Pay
4. Pay with Google - 僅支援 Android Mobile Web、Chrome M61 版本後以及 Google Play Services version 11.4.x 版本後
5. Line Pay
6. Payment Request API
